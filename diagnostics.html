What this page can and cannot show

Can show after button click

  • Public IP address seen by the Vercel edge function
  • Country/region/city hints when Vercel provides them
  • Browser, language, platform, timezone
  • Screen size, viewport, pixel ratio, color depth
  • Approximate RAM and logical CPU cores if browser supports it
  • Connection hints such as effective type and RTT if browser supports it

Cannot show from a normal website

  • Files on the computer
  • Passwords, cookies from other sites, private documents
  • Windows username or local accounts
  • Exact CPU model, motherboard, serial numbers
  • Installed programs and running processes
  • Full disk, antivirus, or system configuration

A real admin dashboard that receives reports from many users needs an authenticated backend and database. The current version is intentionally local and consent-based.
